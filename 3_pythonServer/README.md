### python pyqt5 client

in Arch linux, i did this:
i used the AUR helper paru and pip
```zsh
paru -S pyqt5
pip3 install opcua-client numpy pyqtgraph
```

go to 
```zsh
cd /home/$USER/.local/bin
```
and find the opcua-client binary file
